# waza

A Next.js project featuring a personal portfolio and notes collection.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- MDX-based notes and blog posts
- Responsive design with Tailwind CSS
- Storybook for component development
- Vitest for testing

## Development

```bash
# Run development server
pnpm dev

# Build for production
pnpm build

# Run tests
pnpm test

# Run Storybook
pnpm storybook
```
